# Voice Notes AI

A modern, cross-platform desktop application that turns voice recordings into structured summaries and action items using AI.

## Features
- **Record**: High-quality voice recording.
- **Transcribe**: Converts speech to text automatically.
- **Analyze**: Generates key points and actionable tasks.
- **Local-First**: Runs on your desktop (Windows).

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS
- **Backend**: Node.js, Express
- **Desktop**: Electron
- **AI**: OpenAI / Whisper
